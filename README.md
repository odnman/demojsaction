# Demo javascript action
This action prints "Hello ODNMAN" or "Hello" + the name of a person

## Inputs
### `name-of-you`

**Required** The name of the You. Default `"ODNMAN"`.

## Outputs

### `time`

The time of the message.

## Example usage
uses: odnman/demojsaction@v1.1
with:
    name-of-you: 'Bill'
